# Website Audit Sample

This repository contains a sample website audit
demonstrating how I review small business websites
for SEO, performance, and usability issues.

## Audit Areas
- Page speed and performance
- Mobile responsiveness
- SEO basics (titles, headings, meta descriptions)
- Contact information clarity
- User experience issues

## Notes
This is a sample audit used to demonstrate
the type of feedback and improvements provided to clients.
